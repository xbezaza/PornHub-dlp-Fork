## 🚀 Особенности версии 2.0.3
1. **Исправлена ошибка HTTP 412 (Cloudflare):** Добавлена поддержка заголовков `User-Agent`, `Referer` и `Sec-Fetch-Mode`.
2. **Поддержка Cookies:** Теперь можно скачивать видео, требующие авторизации или обхода CAPTCHA, через файл `cookies.txt`.
3. **Стабильность:** Исправлены пути к `ffmpeg` и `yt-dlp`, добавлен автоматический перезапуск загрузки при сбоях.
4. **Брендинг:** Обновлен интерфейс и метаданные исполняемого файла.

📥 Установка и использование

Вариант 1: Исполняемый файл Windows (Рекомендуется)
1. Скачайте последний релиз `.exe` из раздела [Releases](https://github.com/xBezaza/PornHub-dlp-Fork/releases).
2. Распакуйте архив в любую папку.
3. **(Важно)** Для скачивания защищенных видео положите файл `cookies.txt` в ту же папку, где лежит `.exe`.
   - *Как получить cookies:* Используйте расширение браузера [Get cookies.txt LOCALLY](https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc).
4. Запустите `pornhub-dlp.exe`. При первом запуске будут скачаны зависимости (`yt-dlp`, `ffmpeg`).
5. Вставьте ссылки в поле ввода и нажмите "Скачать".

Вариант 2: Запуск из исходного кода (Python)
Требует Python 3.10+.

1. Клонируйте репозиторий:

git clone https://github.com/xBezaza/PornHub-dlp-Fork.git
cd PornHub-dlp

2. Создайте виртуальное окружение:

python -m venv .venv
# Windows
.venv\Scripts\activate

# Linux/Mac
source .venv/bin/activate

3. Установите зависимости:

pip install -r requirements.txt

4. Запустите приложение:

python main.py

⚙️ Настройка Cookies

Чтобы избежать блокировок Cloudflare и скачивать приватные видео:
1. Откройте браузер в режиме Инкогнито.
2. Войдите в свой аккаунт Pornhub.
3. Перейдите на страницу https://www.pornhub.com/robots.txt.
4. Экспортируйте cookies через расширение в файл cookies.txt.
5. Положите файл в корневую папку программы.
6. В файле Settings.json убедитесь, что есть строка: "cookies_path": "cookies.txt".


🛠 Сборка для Windows
Если вы хотите собрать .exe самостоятельно:
1. Активируйте виртуальное окружение.
2. Перейдите в папку Build/Windows.
3. Запустите build.bat.
4. Готовый файл появится в Build/Windows/Release.
